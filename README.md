# 5060 AI Senior Care App

Emotional support chat and safety-focused assistant service for older adults.

## Getting started

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev`: start local development server
- `npm run build`: build for production
- `npm run preview`: preview production build locally

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
